# 🌱 CarbonCampus

Projet universitaire de calcul d’empreinte carbone réalisé en HTML, CSS et JavaScript.

## 📌 Fonctionnalités

- Calcul carbone du transport
- Calcul carbone de l’alimentation
- Calcul carbone du numérique
- Graphique des résultats avec Chart.js
- Sauvegarde des données avec localStorage
- Utilisation d’API ImpactCO2
- Navigation entre plusieurs pages

---

## 🛠 Technologies utilisées

- HTML5
- CSS3
- JavaScript
- XMLHttpRequest
- Chart.js
- Git / GitHub

---

## 🌍 APIs utilisées

### Transport
https://impactco2.fr/api/v1/transport

### Alimentation
https://impactco2.fr/api/v1/alimentation?category=group&language=fr

### Numérique
https://impactco2.fr/api/v1/thematiques/ecv/1?detail=0&language=fr

---

## 📊 Exemple de calcul

- Transport : distance × émission CO₂
- Aliment : quantité × ECV
- Numérique : usage × ECV
